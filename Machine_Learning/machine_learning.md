# 机器学习笔记

## 本质
让机器去学习：设计一种算法，让算法能够提取中数据所蕴含的规律，这就叫机器学习。如果输入机器的数据是带有标签的，就称作有监督学习。如果数据是无标签的，就是无监督学习。

## 学习方式
### 监督
全部都有标签
- 常用：分类、回归
  - 支持向量机（SVM）
  - 朴素贝叶斯
  - 逻辑回归
  - K近邻（KNN）
  - 决策树
  - 随机森林
  - 线性判别分析（LDA）
  - 大多数的深度学习
  
#### 步骤
- 数据集的创建和分类
- 数据增强（图像旋转，平移，颜色变换，裁剪，仿射变换等）
- 特征工程（特征提取和特征选择），CNN卷积神经网络本身就是一种特征提取和选择的引擎。研究者提出的不同的网络结构、正则化、归一化方法实际上就是深度学习背景下的特征工程。
- 构建预测模型和损失。构建合适的预测模型得到对应输入的输出，保证模型的输出和输入标签的一致性，需要构建模型预测和标签之间的损失函数，常见的损失函数(Loss Function)有交叉熵、均方差等。通过优化方法不断迭代，使模型从最初的初始化状态一步步变化为有预测能力的模型的过程，实际上就是学习的过程。
- 训练。优化方法最常见的就是梯度下降法及其变种，使用梯度下降法的前提是优化目标函数对于模型是可导的。
- 验证和模型选择
- 测试及应用部署

#### 评价指标
            |   Actual P    |   Actual N    |
------------|---------------|---------------|
Predicted P |      TP       |      FP       |
------------|---------------|---------------|
Predicted N |      FN       |      TN       |

TP：预测为P，实际为P，预测正确  
FP：预测为P，实际为N，预测错误  
FN：预测为N，实际为P，预测错误  
TN：预测为N，实际为N，预测正确  

P ： Precision（精确率） = TP / (TP + FP)，预测为P的样本中有多少真实为P（减少误诊）  
R ： Recall （召回率）= TP / （TP + FN），真实P样本中有多少被正确检出（减少漏诊）  
Specificity （特异度）= TN / （TN + FP ），真实N样本中有多少被正确排除  
F1 = 2*P*R / P + R ，平衡精确率与召回率的综合指标  

#### 监督学习、代价函数、激活函数
输入数据 → 网络前向传播（含激活函数） → 输出预测 → 代价函数计算误差 → 反向传播（梯度流经激活函数） → 参数更新  
梯度 = ∂代价函数/∂输出 × ∂激活函数/∂输入  
激活函数、代价函数均非负。

### 非监督
有数据集，无标签
- 常用：关联规则学习、聚类
  - Apriori
  - K-Means

### 半监督
部分标记，部分未标记。用于预测。
- 常用：分类、回归。（可以理解是监督算法的延伸，对已有进行数据建模，在此基础上进行预测）
  - 图论推理算法
  - 拉普拉斯支持向量机（Laplacian SVM）

### 弱监督
标签不一定是对的，不可靠。可能标记不正确、多种标记、标记不充分、局部标记。
已知数据和其一一对应的弱标签，训练一个智能算法，将输入数据映射到一组更强的标签的过程。标签的强弱指的是标签蕴含的信息量的多少，比如相对于分割的标签来说，分类的标签就是弱标签。
